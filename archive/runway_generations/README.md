# Runway Generations Archive

This folder marks the permanent archive location for generated Runway video artifacts and related result JSON files.

Recommended local companion folder:
- `archive/runway_generations/local_exports/`

Notes:
- Generated MP4 files are created locally first and must be copied or moved here intentionally if you want them preserved.
- Do not rely on `outputs/runway_video_run/downloads/` as a permanent store; treat it as a working folder.
- Result JSON files should be saved alongside each generated clip set for traceability.
